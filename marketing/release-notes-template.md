# 米果智读 Release Notes 模板

> 适用：HarmonyOS / iOS 中国区 / Android（华为应用市场为主）
> 字符限制：华为 ≤ 500 字 / Apple ≤ 4000 字符
> 语言：必须提供 zh-Hans，建议同时提供 en-US 用于全球版同步

---

## 模板：标准版（次要更新）

```
v{VERSION}（{YYYY-MM-DD}）

【新增】
- {功能 A 简述（≤ 25 字）}
- {功能 B 简述}

【优化】
- {体验改进点 A}
- {体验改进点 B}

【修复】
- 修复 {场景} 下 {问题描述}
- 修复 {场景} 下偶现崩溃

感谢你的反馈，让米果越来越懂你。
```

---

## 模板：重大版本（含新模块）

```
v{VERSION}（{YYYY-MM-DD}） — {版本主题，例如 "暑期阅读季"}

🌟 重磅新功能
- {模块 A 名称}：{一句话价值描述}
- {模块 B 名称}：{一句话价值描述}

✨ 体验升级
- {场景}：{改进点}
- {场景}：{改进点}

🛠 稳定性
- 修复 {高频问题}
- 提升 {场景} 性能 {数字} %

📚 内容更新
- 新增 {数量} 部公版书籍
- 主题书单：{书单名称}

期待你的反馈：support@readmigo.cn
```

---

## 模板：紧急修复（Hotfix）

```
v{VERSION}（{YYYY-MM-DD}）

紧急修复：
- 修复部分用户在 {场景} 下无法 {操作} 的问题
- 修复极少数情况下 {模块} 闪退

强烈建议立即升级。
```

---

## 文案规范

| 维度 | 要求 |
| ---- | ---- |
| 主语 | 不带 "我们"、"用户"，直接描述功能动作 |
| 动词 | 用 "新增 / 优化 / 修复 / 升级"，禁用 "解决 / 处理" |
| 数字 | 改进幅度可量化时一定写出来（"启动速度提升 30%"） |
| 表情 | 中文版可用🌟✨🛠📚 等点缀，每段不超过 1 个，不连用 |
| 敏感词 | 禁用 "翻墙、VPN、最佳、第一" 等违规词；AI 相关功能用 "智能" 替代亦可 |
| 内购 | 涉及付费功能必须明确 "需 Premium 订阅" |
| 链接 | 仅可放官网/客服邮箱，不放第三方下载链接 |

---

## 多语言对照表（核心动词）

| 中文 | 英文 |
| ---- | ---- |
| 新增 | Added |
| 优化 | Improved |
| 修复 | Fixed |
| 升级 | Upgraded |
| 重磅新功能 | What's new |
| 紧急修复 | Hotfix |
| 期待反馈 | We'd love your feedback |

---

## 发版流程联动

1. 开发完成后由 release-manager 起草 release notes
2. 产品经理 review（重点：是否触发应用市场敏感词、是否需要更新隐私实践）
3. 翻译至 15 种 AppLanguage（参见全局规范），其中 zh-Hans 为主控
4. 提交至 AGC / App Store Connect 时同步上传
5. 发布后同步到：官网博客、微信公众号、用户邮件 (Loops)、社群推送

---

## 历史版本归档

发版后请把最终文案 append 到 `marketing/release-notes-history.md`，便于检索和文案复用。

---

## Pre-publish checklist

- [ ] 字数符合各市场限制
- [ ] 无错别字、无敏感词
- [ ] 多语言版本齐全
- [ ] 与本版本实际改动一致（无虚假宣传）
- [ ] 涉及付费功能已明确订阅要求
- [ ] 链接可访问且为已备案域名
