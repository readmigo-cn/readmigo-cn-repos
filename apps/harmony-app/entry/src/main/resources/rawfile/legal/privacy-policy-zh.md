# 米果智读 隐私政策

> 版本号：v1.0
> 生效日期：2026 年 5 月

**北京瑞光科技有限公司**（以下简称"公司"、"我们"）深知个人信息对您的重要性，承诺按照法律法规要求保护您的隐私。请您仔细阅读本隐私政策，特别是加粗内容。

## 一、我们收集的信息

| 类型 | 用途 | 是否必需 |
|------|------|---------|
| 手机号 / 邮箱 | 账号注册、登录验证 | 是 |
| 设备型号、系统版本 | 适配与问题排查 | 是 |
| 阅读进度、笔记、生词 | 多端同步与个性化推荐 | 是 |
| 订阅与支付记录 | 提供订阅服务 | 是 |
| 网络信息（IP、运营商） | 安全风控、异常检测 | 是 |
| 崩溃与性能日志 | 修复 bug、性能优化 | 否（可关闭） |
| 行为统计（页面浏览、点击） | 改进产品体验 | 否（可关闭） |

## 二、我们不会收集

- 通讯录、短信、通话记录
- 精确地理位置（GPS）
- 相册（除您主动选择上传头像外）
- 麦克风录音（除您主动开启朗读评测外）

## 三、我们如何使用您的信息

3.1 提供并维护服务（账号、阅读、同步、订阅）。
3.2 通过推送通知向您发送学习提醒、新书上架、活动信息（您可在系统设置关闭）。
3.3 根据法律法规要求向监管部门提交相关信息。
3.4 在去除个人标识后用于产品改进和数据分析。

## 四、我们如何共享您的信息

4.1 我们 **不会** 向任何第三方出售您的个人信息。
4.2 在以下情形下我们可能共享您的信息：
   - 经您明确同意
   - 法律法规要求或司法机关依法要求
   - 为提供 AI 翻译 / TTS 服务，向 AI 服务商传递必要文本（不含您的身份信息）
   - 为提供支付服务，向支付平台传递订单金额（不含您的阅读内容）
4.3 第三方 SDK 列表见附录。

## 五、您的权利

您有权：

- **查询**：随时查看您的个人信息
- **更正**：修改不准确的信息
- **删除**：删除您的笔记、生词或阅读记录
- **注销**：永久删除账号及所有数据（设置 → 账号 → 注销）
- **撤回授权**：关闭可选权限

## 六、信息存储与安全

6.1 您的数据存储在中国境内的云服务器（华为云、阿里云）。
6.2 我们采用 HTTPS 传输、AES 加密存储、严格访问控制等安全措施。
6.3 信息保留期：账号注销后 30 天内删除；备份数据 90 天内永久删除。

## 七、未成年人保护

7.1 我们严格遵守《未成年人保护法》和《个人信息保护法》中关于未成年人的规定。
7.2 如您未满 14 周岁，请在监护人同意下使用本服务。
7.3 详见《未成年人保护规则》。

## 八、Cookie 和类似技术

我们使用本地存储（KV Store、Preferences）保存登录状态、阅读进度、用户偏好。您可在系统设置中清除应用数据。
