# 米果智读 截图规范（Screenshot Spec）

> 版本：v1.0
> 适用范围：HarmonyOS NEXT 应用市场 / iOS App Store 中国区
> 输出脚本：`apps/harmony-app/scripts/screenshots/generate-screenshots.sh`

---

## 1. 设备与分辨率矩阵

| 平台 | 设备类型 | 截图分辨率（px） | 备注 |
| ---- | -------- | ---------------- | ---- |
| HarmonyOS | 手机（直板） | 1260×2720 | 华为 Mate 60 系列 |
| HarmonyOS | 折叠屏（展开） | 2208×2480 | Mate X5 展开态 |
| HarmonyOS | 平板（横屏） | 2560×1600 | MatePad Pro 13.2 |
| iOS CN | 6.9 英寸 iPhone | 1290×2796 | iPhone 15 Pro Max（必传） |
| iOS CN | 6.5 英寸 iPhone | 1284×2778 | iPhone 14 Plus（必传） |
| iOS CN | iPad Pro 12.9 | 2048×2732 | 选传，提升通过率 |

**多语言要求**：
- 必传：简体中文（zh-Hans）
- 选传：英文（en-US），用于全球 iOS 双区共用素材

## 3. 视觉规范

| 元素 | 规范 |
| ---- | ---- |
| 主色 | 品牌橙 `#F26B1F` + 中性白 `#FFFFFF` |
| 字体 | 标题：HarmonyOS Sans SC Bold；正文：HarmonyOS Sans SC Regular |
| 文字位置 | 顶部 1/4 区域，避开状态栏与刘海 |
| 状态栏 | 必须替换为 demo 数据：时间 9:41、电量 100%、信号满格、运营商空白 |
| 安全区 | 顶部留 88pt、底部留 34pt 内容缓冲 |
| 图层结构 | PNG 24-bit，无 alpha；DPI 视设备而定 |

## 4. 输出与命名

```
marketing/screenshots/
├── harmony/
│   ├── 1-zh-Hans.png
│   ├── 1-en-US.png
│   └── ...
└── ios-cn/
    ├── 1-zh-Hans.png
    └── ...
```

命名格式：`{序号}-{locale}.png`，序号 1-10 与本文档一一对应。
