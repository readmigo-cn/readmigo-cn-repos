# 生成式 AI 服务备案 SOP

> 依据《生成式人工智能服务管理暂行办法》(2024)
> 创建时间：2026-04-26

---

## 1. 备案要求概览

### 1.1 适用法规

- 《生成式人工智能服务管理暂行办法》(国家互联网信息办公室，2024 年施行)
- 《互联网信息服务管理办法》
- 《网络安全法》
- 《数据安全法》
- 《个人信息保护法》

### 1.2 备案主体

**要求**: 在中华人民共和国境内注册的企业法人

**决策**: 需确认阅可 (Readmigo CN) 的运营主体公司
- 选项 A: 新设公司 (推荐，独立法人实体)
- 选项 B: 现有公司 (如有)

### 1.3 备案时间

| 阶段 | 时间要求 | 说明 |
|------|----------|------|
| **上线前** | 应用上架前 4-8 周 | 必须完成备案才能上线 |
| **变更后** | 重大变更后 10 个工作日内 | 如 LLM provider 变更、功能重大调整 |
| **年度** | 每年第一季度 | 年度报告 |

## 2. 备案材料清单

### 2.1 基础资质材料

- [ ] **营业执照** (扫描件，加盖公章)
- [ ] **法人身份证** (正反面扫描件)
- [ ] **经办人身份证** (正反面扫描件 + 授权委托书)
- [ ] **ICP 备案截图** (已完成 ICP 备案的证明)
- [ ] **网络安全等级保护备案证明** (等保 2.0，建议二级)

### 2.2 技术材料

- [ ] **服务基本信息表**
  - 服务名称：阅可 (Readmigo)
  - 服务形式：移动应用 (HarmonyOS)
  - 服务领域：英语阅读 + AI 学习助手
  - 用户规模预测：首年 10 万用户
  
- [ ] **算法备案信息**
  - 算法名称：英语阅读 AI 助手算法
  - 算法类型：生成式人工智能 (AIGC)
  - 技术路线：基于大语言模型 (LLM) 的文本生成
  - 训练数据来源：
    - 公版书内容 (Project Gutenberg, Standard Ebooks)
    - 国内公版内容 (CTEXT, Wikisource, 书格)
    - 不用于模型训练，仅用于推理 (inference)
  
- [ ] **LLM 提供商信息**
  - 主要提供商：DeepSeek (深度求索)
  - 备选提供商：Qwen (阿里云通义千问)、Zhipu (智谱 AI)、Wenxin (百度文心一言)
  - 提供商资质：均为中国境内注册企业
  - 合作协议：与每家 LLM 提供商签署 API 使用协议
  
- [ ] **技术架构图**
  ```
  阅可 APP (HarmonyOS)
       ↓
  后端 API (server-cn, NestJS)
       ↓
  LLM 适配层 (@readmigo-cn/llm-adapter)
       ↓
  LLM 提供商 API (DeepSeek / Qwen / Zhipu / Wenxin)
  ```

### 2.3 内容安全材料

- [ ] **内容审核制度**
  - 审核流程：AI 生成内容 100% 标识
  - 审核人员：专职内容审核员 2 名
  - 审核记录：保存至少 6 个月
  
- [ ] **AI 生成内容标识方案**
  - 标识位置：所有 AI 生成内容末尾添加"由 AI 生成"标识
  - 标识样式：灰色小字，字号 10pt
  - 用户提示：首次使用时弹窗告知"AI 生成内容可能不准确"
  
- [ ] **内容白名单制度**
  - 公版书白名单：首批 500 本公版书 (来源：Project Gutenberg + Standard Ebooks)
  - 审核标准：
    - 无政治敏感内容
    - 无色情低俗内容
    - 无暴力恐怖内容
    - 无宗教极端内容
  - 审核流程：人工初审 + 复审
  
- [ ] **用户举报机制**
  - 举报入口：APP 内"反馈"功能
  - 处理时限：24 小时内响应
  - 处理流程：收到举报 → 人工审核 → 下架问题内容 → 反馈用户

### 2.4 数据安全材料

- [ ] **数据安全管理制度**
  - 数据分类：用户数据、内容数据、日志数据
  - 存储位置：华为云 GaussDB (中国境内)
  - 备份策略：每日增量备份 + 每周全量备份
  - 访问控制：仅授权技术人员可访问
  
- [ ] **个人信息保护制度**
  - 隐私政策：符合《个人信息保护法》要求
  - 用户同意：首次启动 APP 时弹窗征得同意
  - 数据最小化：仅收集必要信息 (账号、阅读进度)
  - 用户权利：支持查询、更正、删除个人信息
  
- [ ] **数据出境说明**
  - 数据不出境：所有数据存储在境内 (华为云)
  - LLM 调用：用户问题传输至 LLM 提供商 API (境内服务器)
  - 传输加密：HTTPS + AES-256 加密

### 2.5 应急处置材料

- [ ] **应急处置预案**
  - 应急小组：3 人 (技术负责人 + 产品负责人 + 法务)
  - 响应时限：发现违规内容后 2 小时内下架
  - 处置流程：
    1. 收到监管通知/用户举报
    2. 立即下架问题内容
    3. 人工复核全部相关内容
    4. 向监管报告处置结果
  
- [ ] **7x24 小时值班制度**
  - 值班人员：2 名轮班
  - 联系方式：值班电话 + 微信工作群
  - 值班记录：每日值班日志

## 3. 备案流程

### Step 1: 准备阶段 (Week 1-2)

1. **确认运营主体**
   - 注册新公司或指定现有公司
   - 办理营业执照 (如新注册)
   
2. **完成 ICP 备案**
   - 通过华为云备案系统提交
   - 时间：2-4 周
   
3. **完成等保备案**
   - 委托第三方测评机构进行等保测评
   - 时间：2-3 周

### Step 2: 材料准备 (Week 3-4)

1. **撰写制度文件**
   - 内容审核制度
   - 数据安全管理制度
   - 个人信息保护制度
   - 应急处置预案
   
2. **准备技术材料**
   - 技术架构图
   - 算法说明
   - LLM 提供商合作协议
   
3. **准备资质材料**
   - 营业执照扫描件
   - 法人/经办人身份证
   - ICP 备案截图

### Step 3: 提交备案 (Week 5)

1. **登录备案系统**
   - 系统：国家网信办生成式 AI 服务备案系统
   - 网址：https://beian.cac.gov.cn (示例)
   
2. **填写备案信息**
   - 服务基本信息
   - 算法信息
   - LLM 提供商信息
   
3. **上传材料**
   - 基础资质材料
   - 技术材料
   - 内容安全材料
   - 数据安全材料
   - 应急处置材料

### Step 4: 等待审核 (Week 6-10)

- **审核时限**: 4-8 周
- **审核流程**:
  1. 形式审查 (5 个工作日)
  2. 实质审查 (15 个工作日)
  3. 专家评审 (10 个工作日，如需)
  4. 审批决定 (5 个工作日)
  
- **可能的补正**: 如材料不全，需在 10 个工作日内补正

### Step 5: 获得备案 (Week 11-12)

- **备案凭证**: 《生成式人工智能服务备案证明》
- **备案编号**: 格式如"网信办备〔2026〕第 XXXX 号"
- **有效期**: 3 年

## 4. 备案后义务

### 4.1 日常运营

- [ ] **内容审核记录**: 保存至少 6 个月
- [ ] **用户举报处理**: 24 小时内响应
- [ ] **数据安全监测**: 每日巡检
- [ ] **值班日志**: 每日记录

### 4.2 变更报告

| 变更类型 | 报告时限 | 报告方式 |
|----------|----------|----------|
| LLM 提供商变更 | 变更后 10 个工作日 | 备案系统提交变更 |
| 服务功能重大调整 | 变更前 10 个工作日 | 重新备案 |
| 运营主体变更 | 变更前 30 个工作日 | 重新备案 |
| 法人变更 | 变更后 10 个工作日 | 备案系统更新 |

### 4.3 年度报告

- **时间**: 每年第一季度
- **内容**:
  - 上年度服务运营情况
  - 用户规模、活跃度
  - 内容审核情况
  - 安全事件及处置
  - 本年度发展计划

## 5. 关键风险与缓解

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|----------|
| 备案延期 (超过 8 周) | 高 | 上线延期 | **Phase 0 Day 1 启动**，预留 12 周 |
| 公版书内容未通过 | 中 | 部分下架 | 白名单 + 渐进上架，首批 500 本保守策略 |
| LLM 提供商资质问题 | 低 | 需更换 provider | 选择头部厂商 (DeepSeek/Qwen/智谱/百度) |
| 数据出境合规风险 | 低 | 监管处罚 | 明确数据不出境，LLM API 调用走境内节点 |
