## 第二条 服务内容与范围

### 2.1 服务描述

米果智读是一款英文学习应用，为用户提供以下核心服务：

| 服务模块 | 内容说明 |
|---------|--------|
| **英文图书阅读** | 提供海量英文书籍（公版书），支持同步进度、书签、笔记功能 |
| **AI 划词服务** | 划选单词/短语，AI 提供中英释义、发音、例句 |
| **AI 翻译与解释** | AI 生成段落翻译、文章摘要、阅读问答 |
| **语音朗读（TTS）** | 支持真人级TTS朗读，可调节语速和音色 |
| **词汇学习** | 收集生词本、定期复习、词汇测试 |
| **阅读数据同步** | 跨设备同步阅读进度、笔记、生词本 |
| **订阅服务（Pro）** | 高级功能需订阅，详见第七条 |

### 2.2 服务地域

本服务仅向中华人民共和国境内用户提供，用户设备需为HarmonyOS NEXT系统。

### 2.3 服务形式

本应用为移动端原生应用，通过华为应用市场（AppGallery）下载安装，不提供网页版本。
