## 采购渠道建议

### 推荐渠道排名

| 渠道 | 优势 | 劣势 | 适用情形 |
|------|------|------|---------|
| **华为官方商城**（企业版） | 1. 原装品质 100%保证 2. 企业采购折扣（3-8%） 3. 含税票 + 售后优先 4. "先享后付"支持 | 页面更新慢，部分新品暂缺 | 首选，尤其旗舰机 |
| **京东企业购** | 1. SKU 全 2. 送达快（2-3 天） 3. 支持发票 + 报销 4. 售后无忧 | 价格与官方无本质区别 | 备选，时间紧急时选用 |
| **苏宁 / 国美线上** | 1. 竞争价格 2. 企业采购渠道 | SKU 不及京东 | 可货比三家，折扣力度好时选用 |
| **授权经销商** | 1. 可能有现货 2. 有时有活动优惠 | 售后可能有滞后 | 急需时可考虑 |

### 采购建议

1. **尽早锁定大订单**
   - 联系华为官方商务（华为有企业采购团队）
   - 提交采购清单 + 预算，协商折扣
   - 预期折扣：3-8%（视订单规模）

2. **充分利用折扣政策**
   - 华为"先享后付"：可 30-45 天结算（现金流优化）
   - 企业批采：10 台以上可获企业价

3. **新品延期应对**
   - Pura 70 等新品可能延期（关注官方发布日期）
   - 备选方案：先采购 P70，后期补充 Pura 70 或 Mate 60 Pro 第二部

4. **供应链冗余**
   - 关键开发机（Mate 60 Pro）采购 2 部，防止一部故障影响进度
   - 平板 / 折叠屏延期采购（不在关键路径上）

## 硬件设备到货与接收流程

### 到货与检验（W2-W3）

```
1. 签收 → 核对清单（PN 码、数量、包装完整度）
2. 初检 → 开箱、检查屏幕、拨打试通话、充电测试
3. 激活 → 华为账号登录（使用公司统一账号）
4. 贴标 → 背面贴编号标签（防丢失）
5. 入库 → 记录序列号到 Sheet，存储柜归档
```

### 设备编号规范

**格式**：`HW-[型号简称]-[序号]-[部门缩写]`

示例：
- `HW-MATE60-01-DEV`（Mate 60 Pro，开发部，序号 01）
- `HW-MATEX5-01-QA`（Mate X5，QA 部，序号 01）
- `HW-IPAD13-01-DEV`（MatePad Pro 13.2，开发部，序号 01）

### 设备责任制

| 设备类型 | 分配原则 | 责任人 | 备注 |
|---------|--------|--------|------|
| **Mate 60 Pro** | 分配给 HarmonyOS 工程师个人（1 台） | 工程师本人 | 日常开发机，个人签名管理 |
| **P70** | 测试库存，不分配 | QA 负责人 | 用于多轮测试，集中存放 |
| **Pura 70** | 分配给 HarmonyOS 工程师个人（备用） | 工程师本人 | 防止主力机故障的备用机 |

1. **故障报告** → 拍照记录 + 填写故障单
2. **华为售后预约** → 通过华为官方 App / 售后热线
3. **维修或换机** → 记录周期与成本，反馈采购
4. **费用认定** → 正常故障（保修期内）由华为负责；人为损坏由使用者赔付

## 关键采购决策

### Q: 为什么不采购中低端机？

**A**: 
- Readmigo 定位英语学习中高端用户，目标用户使用旗舰 / 高端机
- 中端机采购两部（P70 + Pura 70），已覆盖"入门高端"场景
- 低端机（<2000 元）使用者极少，延期采购

### Q: 为什么必须有两部 Mate 60 Pro？

**A**:
- 一部个人开发机，一部参考 / 对标机（防止一部故障影响产品定义）
- 测试速度加快（并行测试不同分支）
- 初期不能丢失开发效率

### Q: Mate X5 折叠屏这么贵（13K），真的需要？

**A**:
- 折叠屏是 HarmonyOS NEXT 核心创意场景，产品必须支持
- 早期适配能发现未知问题，晚期适配成本 2-3 倍
- 可在 W5-W12 采购，不在关键路径上

### Q: 存储柜 / 显示器有必要买？

**A**:
- **存储柜** ✓ 必需：防损失、统一充电、提高查找效率
- **显示器** ✗ 可选：只有调试性能时需要（初期无此需求）
