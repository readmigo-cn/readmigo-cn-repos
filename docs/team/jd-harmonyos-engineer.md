## 主要职责

### 核心开发任务

1. **ArkTS / ArkUI 原生应用开发**
   - 使用 ArkTS 开发 UI 组件和页面逻辑
   - 遵循 HarmonyOS Design 设计规范
   - 性能优化（帧率 60fps+，冷启动 < 500ms）
   - 界面适配（手机 / 平板 / 折叠屏）

2. **C++ NAPI 桥接开发与维护**
   - 设计并实现本地能力与应用层接口
   - 完善已有 NAPI 模块（国产 LLM 推理、TTS、图像处理）
   - 性能基准测试与优化
   - 内存泄漏排查与工具链集成

3. **HMS 服务集成**
   - HMS Core（推送、分析、支付、身份认证）
   - 华为云服务对接（华为云存储、CDN）
   - Huawei IAM 和 OAuth 登录集成
   - 合规上报（隐私、权限）

4. **架构设计参与**
   - MVVM + Clean Architecture 的执行
   - 状态管理方案设计（建议 GetX 或 River Pod 思路）
   - 离线优先与数据同步策略
   - 错误处理与监控体系

5. **性能与稳定性**
   - Profiler 使用与性能分析
   - Crash 预防与 Sentry 集成
   - 内存、CPU、电池使用优化
   - 单元 / 集成测试编写（目标覆盖 > 70%）

## 常见问题

### Q: 是否必须在北京？

**A**: 首选北京或周边（一小时车程内）。若远程且参与度高，可考虑（但初期需 1-2 周驻北京）。

### Q: 股权给多少？

**A**: 取决于加入时机与贡献：
- 第 1 人（早期风险）：0.8%-2%
- 第 2-3 人（产品-市场契合期）：0.5%-1.2%
- 后续人员：根据融资后重新分配

### Q: 多久能升职 / 调薪？

**A**: 
- **调薪**：年度考核（Q4）+ 突出贡献即时奖励
- **晋升**：18-24 个月表现突出 → 晋升为 P6 / 技术负责人

### Q: 是否要倒班 / 假期限制？

**A**: 无倒班。假期按国家规定 + 15 天年假。重大版本发布前可能需加班（补休）。

### Q: 对海外代码库（Readmigo）的期望？

**A**: 初期 80% 精力在国内版本，20% 代码同步 / 架构对齐。等国内版本稳定后，可参与海外特性开发。

### Q: 入职后的学习支持？

**A**: 
- HarmonyOS 官方培训课程免费报名
- NAPI 与性能优化由创始人与 C++ 工程师指导
- 技术分享与跨团队学习会（月度）
