## 主要职责

### 核心开发任务

1. **server-cn 项目维护与开发**
   - NestJS 框架下的 REST API 开发与维护
   - 模块化、分层设计（Controller → Service → Repository）
   - 代码质量管理（TypeScript 严格模式、代码审查、单测 > 60%）
   - 版本迭代与向后兼容性管理

2. **华为云服务集成**
   - 华为云 OBS（对象存储）- 书籍、音频、用户头像存储
   - 华为云 CDN - 内容分发加速
   - 华为云 RDS for PostgreSQL / GaussDB - 数据库运维
   - 华为云容器镜像仓库（SWR） - Docker 镜像管理
   - 华为云 ECS 与负载均衡（暂不使用 K8s，单机或小规模）

3. **数据库设计与优化**
   - PostgreSQL / GaussDB 数据库 schema 设计
   - 复杂查询优化与索引策略
   - 数据一致性与事务管理
   - 慢查询分析与瓶颈定位
   - 数据备份与灾难恢复方案制定

4. **国产 LLM 适配层**
   - DeepSeek / 百度文心 / 智谱 GLM / 阿里 Qwen 等 LLM API 接口适配
   - 本地 LLM 部署与 vLLM / Ollama 集成（可选）
   - Token 使用量计费模型设计
   - Prompt 版本管理与 A/B 测试框架

5. **AI 功能 API 设计**
   - 内容推荐引擎 API（协同过滤、向量化推荐）
   - 单词本智能推荐（基于阅读历史、学习进度）
   - 理解力测试 API（自动生成题目、评分、反馈）
   - 个性化学习路径 API

6. **国产 TTS 集成**
   - 讯飞 TTS（主要方案）
   - 对标方案：Microsoft Azure 中国版、腾讯云 TTS
   - 合成队列管理与异步处理
   - 多语言支持（英文、汉语）

7. **系统监控与运维**
   - 日志收集与分析（ELK Stack 或腾讯云日志服务）
   - 性能监控与告警（CPU、内存、数据库连接数）
   - 定期数据库备份验证
   - 生产环境故障应急响应

## 必备要求

### 技术背景

- **开发经验**：3+ 年 Node.js 后端开发，2+ 年 NestJS 或类似框架经验
- **核心语言**：熟练 TypeScript（类型系统、高级类型）
  - 理解装饰器、反射等高级概念
  - 遵循 SOLID 原则、设计模式应用
- **数据库**：深入理解 PostgreSQL / MySQL
  - 索引设计与查询优化（Explain Plan 分析）
  - 事务隔离级别与死锁排查
  - 熟悉 SQL 与 ORM（TypeORM / Prisma）
- **API 设计**：RESTful API 设计规范
  - HTTP 状态码、错误处理、版本管理
  - 分页、搜索、过滤、排序最佳实践
  - 速率限制与并发控制
- **系统设计**：理解微服务 / 模块化架构
  - 缓存策略（Redis）
  - 异步任务队列（Bull / BullMQ）
  - 消息队列基础（可选）
- **容器与部署**：Docker 与基础部署
  - Dockerfile 编写与优化
  - 容器网络与卷管理
  - 基础 CI/CD（GitHub Actions / 华为云 CodePipeline）

### 思维与行动

- **需求理解能力**：能从产品需求、用户故事反推 API 设计与数据库 schema
- **代码质量意识**：积极进行代码审查、乐于接收反馈
- **文档习惯**：API 文档（Swagger）、架构决策记录（ADR）
- **性能优先**：对数据库慢查询、大数据集处理、缓存有敏感度
- **安全意识**：了解 SQL 注入、XSS、CSRF、认证授权基础

## 常见问题

### Q: 是否必须在北京？

**A**: 首选北京或周边（一小时车程）。若远程工作经验丰富且沟通主动，可考虑其他城市（但初期需 2-3 周驻北京）。

### Q: 股权与薪资的比例？

**A**: 薪资 + 股权组合：
- 全薪资方案：30-50K（创业初期，多数人选择）
- 薪资 + 股权方案：25-35K + 0.5%-1.5%（仅限高度认可者）

### Q: 华为云与国产 LLM 我没经验，能学吗？

**A**: 可以！我们提供：
- 华为云官方培训课程（免费）
- ACP 认证补贴
- LLM 基础工作坊（创始人主持）
- 项目中边做边学

但需要你有**快速学习能力**与**主动性**。

### Q: 是否支持远程长期？

**A**: 初期 3-6 个月需定期线下（每月 1-2 周）以建立信任与文化契合度。之后，表现优异者可全远程，但需：
- 每月至少 1 周回北京（sprint 计划 + code review）
- 或参加季度全员 offsite

### Q: 职业发展路线？

**A**: 
- **技术路线**：后端工程师 → 技术负责人 → CTO（18-24 个月）
- **产品路线**：后端 → 产品经理（转向）
- **创业路线**：参与天使轮融资与扩团队（12+ 个月后）

### Q: 入职后如何快速上手？

**A**: 
- **W1**：环境搭建、API 文档阅读、现有代码 walkthrough
- **W2-W3**：修复 bug / 小特性（获得代码库熟悉度）
- **W4+**：独立开发新 API / 数据库优化任务
- **全程**：每日同步 15 分钟，周五 1 小时 code review
